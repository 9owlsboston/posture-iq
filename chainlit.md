# 🛡️ SecPostureIQ

Welcome to **SecPostureIQ** — your ME5 Security Posture Assessment Agent.

I can help you assess your Microsoft 365 E5 security deployment and create a **Get to Green** plan.

## What I Can Do

- **📊 Secure Score** — Check your tenant's Microsoft Secure Score with category breakdowns
- **🛡️ Defender Coverage** — Assess M365 Defender deployment across all workloads
- **📋 Purview Policies** — Review DLP, sensitivity labels, and retention policies
- **🔐 Entra ID Config** — Evaluate Conditional Access, PIM, and Identity Protection
- **📝 Remediation Plan** — Generate a prioritized remediation plan with scripts
- **📈 Adoption Scorecard** — Create an ME5 adoption scorecard with RAG status
- **📖 Get-to-Green Playbook** — Retrieve step-by-step remediation playbooks
- **📊 Fabric Snapshot** — Push posture data to the Fabric lakehouse for dashboarding

## Getting Started

Try asking:
- *"Assess this tenant's ME5 security posture"*
- *"What's our current Secure Score?"*
- *"Show me the Defender coverage gaps"*
- *"Generate a remediation plan for our Entra ID gaps"*
